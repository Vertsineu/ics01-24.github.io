# 成绩

> 成绩各部分占比仅作为参考

- 期中考试：**15%**
- 期末考试：**25%**
- 平时成绩：**50%**
    - 章节课后作业
    - 课程实验
    - LC3汇编器/模拟器
- 坚持与努力、出勤率和参与度(课程/讨论课)、协作能力:**10%**
